※ 初回応答150msは、M3/M2クラス＋軽量モデル（例：SmolLM2‑135M）での温起動時測定の目安です。端末/モデル/量子化/温度で変動します。

※ STUN/TURNはNAT越えのための標準的仕組みで、場合により中継サーバを通過します。データはDTLS‑SRTPおよびアプリ層E2Eで保護され、サーバが内容を復号することはできません。

暗号スイートはNoise Protocol Frameworkに準拠。

Secure Enclaveの詳細はApple公式ドキュメントを参照。

量子計算との関係：AES‑256はGroverによる探索でも約2¹²⁸相当の安全性。現時点の実用上十分な強度というのが専門的コンセンサスです。
