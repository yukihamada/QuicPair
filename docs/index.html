WebRTC P2P接続により、平均150ms以下の初回応答時間を実現。ネットワーク遅延を最小限に抑えた、ストレスフリーな会話体験。

Noise IKプロトコル（Curve25519 + AES-256-GCM）による二重暗号化。WebRTC DTLSに加えて、アプリケーション層でもE2E暗号化。
